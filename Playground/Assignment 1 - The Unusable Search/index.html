The Worst Search System Imaginable

Welcome to the worst portfolio search page you'll ever use! Here’s how to “search”:

  1. Drag one letter to the Tiny Drop Zone.
  2. Press Done to lock in that letter to your current word (the letters will reshuffle – have fun!).
  3. Repeat for each letter in your word.
  4. When you’re ready, click Start Search to see if anything matches your typed word.
  5. If you want to start over (because let’s face it, you will), press Clear Word to reset everything.

Good luck… you’ll need it!
