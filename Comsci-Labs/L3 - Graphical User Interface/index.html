Welcome!

For the Third Lab; GUIs, I created this Etch-A-Sketch drawing simulator. Please read important information below.

Navigation / Drawing

To move the cursor; Grab and spin the dials, Press on the arrows within the dials, or use the ↑ ↓ ← → keys.
Press the S key to shake (clear) the board
Use the the slider to adjust stroke size

Warnings / Disclaimers

This simulator is only designed to function at a standard desktop breakpoint size.
The dials are NOT made using p5’s standard GUI elements; they were instead built using DOM elements, event listeners, and CSS transforms. If these are unsatisfactory for the lab, Please let me know.
